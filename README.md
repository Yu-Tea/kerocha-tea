## 🌈 🫖 ケロチャがお茶の色を選ぶ仕組みについて

### たった3つの質問で、あなた専用のお茶が完成！

ケロチャの質問は、お茶の色を決める3つの要素に対応しています。

🎨 **色相** - 色の種類です。赤・緑・青…など6種類で分類しています  
✨ **彩度** - 色の鮮やかさです。高い・普通・低いで分類しています  
☀️ **明度** - 色の明るさです。明るい・普通・暗いで分類しています  

### なぜ毎回違う色になるの？

「同じ回答をしたのに、さっきと違う色だ！」と思ったあなた、その通り！  
実は、回答に対する色相・彩度・明度は固定値ではなく、一定の幅があるんです。

例えば色相が「緑色」となる回答を選んだとき…  
🍀 黄緑 ←─ この範囲からランダムで決定 ─→ 🌲 緑

このようにして色相・彩度・明度の値を決めるため、その時々で微妙に違う色が選ばれるのです。

**つまり?**  
ケロチャは毎回、その瞬間のあなたにぴったりのお茶を淹れてくれているんです🍵💚
