## 🌈 🫖 ケロチャがお茶の色を選ぶ仕組みについて
### たった3つの質問で、あなた専用のお茶が完成！

ケロチャの質問は、お茶の色を決める3つの要素に対応しています。

🎨 **色相** - 赤？ 青？ 緑？ 色の種類を決めます  
✨ **彩度** - 鮮やかな色？ 落ち着いた色？  
☀️ **明度** - 明るい色？ 暗い色？

### なぜ毎回違う色になるの？
「同じ回答をしたのに、さっきと違う色だ！」と思ったあなた、その通り！  
実は、あなたの回答は**色相・彩度・明度の範囲**を決めているんです。

例えば色相が「緑色」となる回答を選んだとき...  
🍀 黄緑 ←─ この範囲からランダム ─→ 🌲 緑


同じ回答でも、その時々で微妙に違う色が選ばれます。

**つまり?**  
ケロチャは毎回、その瞬間のあなたにぴったりのお茶を淹れてくれているんです🍵💚
